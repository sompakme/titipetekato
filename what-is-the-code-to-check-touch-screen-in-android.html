    What is USSD code running?

    What is a USSD Code Anyway? A USSD code is entered into phones to perform actions. They are mainly used by network operators to provide customers with easy access to pre-configured services, including call-forwarding, balance inquiries and multiple SIM functions.

    What happens when you call ## 002?

    ##002# - If your voice call or data call, or SMS call has been forwarded, dialling this USSD code will erase them. *#21# - By dialling this USSD code, you would get to know if your calls have been diverted somewhere else or not.
